# Runtime Strategy

Fecha: 2026-01-12T18:30:47.813Z

## Runtime Strategy

- Node.js:
  - Frontend (Vite)
  - Supabase Edge Functions
  - Tooling (Vitest, ESLint)

- Deno:
  - Scripts específicos heredados / experimentales
  - No críticos para el flujo de firma

Decision:

- No se migra Deno antes del beta.
- Se reevalúa post-beta según uso real.

Nota: Este documento neutraliza la crítica sobre la convivencia de runtimes y deja la decisión explícita para la fase post-beta.
