## Propósito

Este directorio contiene la **arquitectura de messaging** de EcoSign. Su objetivo es asegurar que toda la comunicación (landing, docs, emails, UI, marketing) comunique de forma consistente el valor real del producto:

> **EcoSign no vende firmas digitales. EcoSign vende protección del trabajo sensible.**

## Resumen Ejecutivo

### Posicionamiento
| Dimensión | Posicionamiento |
|-----------|-----------------|
| **Categoría** | Protección del trabajo sensible |
| **Diferenciador** | Evidencia verificable sin exponer contenido |
| **Audiencia** | Profesionales que trabajan con documentos sensibles |
| **Promesa** | No esperes a que pase lo peor. Protegelo hoy. |

### Mensajes Clave
| Nivel | Mensaje |
|-------|---------|
| **Principal** | "Protegé tu trabajo, no solo documentos." |
| **Soporte** | "Protegés sin exponer contenido." |
| **Táctico** | "Pegás una lista de mails. EcoSign hace el resto." |

### Tonos de Voz
| Contexto | Tono |
|----------|------|
| **Hero / Homepage** | Directo, emocional, sin tecnicismos |
| **Features** | Claro, concreto, sin hype |
| **Documentación** | Preciso, verificable, sin opacidad |
| **Error / Alertas** | Útil, sin culpa, con salida |

### Palabras Clave
| ✅ USAR | ❌ EVITAR |
|---------|----------|
| Proteger, evidencia, verificable, trazabilidad | Firma digital, e-signature, DocuSign |
| Huella, hash, sello de tiempo, anclaje | Blockchain, Web3, descentralizado (sin contexto) |
| Antes, después, cuándo, continuidad | Inhackeable, 100% seguro, infalible |

## Tests de Calidad

Antes de publicar cualquier mensaje, aplicá estos tests:

### Test de los 3 Segundos
¿Un usuario entiende el valor en 3 segundos?

### Test de DocuSign
¿Este mensaje podría aplicarse a DocuSign? (Si sí, descartar.)

### Test de la Madre
¿Tu madre (o un abogado de 50 años) entiende el mensaje?

### Test del Perito
¿Un perito puede verificar técnicamente esta afirmación?

## Referencias Externas

### Marcas con Buen Messaging
- **Basecamp:** Claridad. Sin hype. Directo al valor.
- **Hey.com:** Posicionamiento fuerte. "Email para gente ocupada."
- **Proton:** Privacidad como feature central. Sin tecnicismos.
- **Fastmail:** "Email como debería ser." Simple, claro.

### Marcas a Evitar
- **DocuSign:** Commodity. "Firmas digitales."
- **Adobe Sign:** Genérico. "Firma electrónica segura."
- **Crypto startups:** Hype. "Web3 descentralizado blockchain."
