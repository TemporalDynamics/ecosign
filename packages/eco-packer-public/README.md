# @temporaldynamics/eco-packer-public

Public API surface for eco-packer. This package exposes types and stub
functions only. It is intended for integrations that need compile-time
contracts and documentation without access to internal implementations.

## Installation

```
npm install @temporaldynamics/eco-packer-public
```

## Usage

```ts
import { pack, verify, unpack } from '@temporaldynamics/eco-packer-public';
```

## Notes

- `pack`, `verify`, and `unpack` are stubs in this public package.
- For production integrations, contact the EcoSign team for access.
